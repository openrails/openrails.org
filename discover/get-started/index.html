Getting Started with Open Rails (version 1.6 or above).


This "Getting Started" describes how to install, download content and start a train driven by a diesel locomotive running on one specific route for Open Rails (abbreviated OR furtheron). Running a train on this route will learn you the basics of Open Rails. Once you've experienced this you definitively want more...

  • Download the Open Rails (1.6 or higher) installer and install in the usual way on a Windows system. You have now installed the OR program. Content to be downloaded in the next step.
  • Start OR. The "Content" form pops up. Highlight the route called "Demo Model 1". The rest of this "Getting started" document will be based on this route. Change the "Install Path" if needed. Diskspace needed for this route is approximately 600 MB. Click button "Install".
  • After the route is installed, press button "OK" to leave the "Content" form.
  • Click button "Start". A "Windows Defender Firewall" window might popup. Allow access for Private networks. OR will start in full screen mode. Keys <Alt + Enter> toggle between full screen and windowed mode. You can always stop OR by pressing the keys <Alt + F4>. This will bring you back into the menu.
  • OR starts with the Pause Menu. Key <Esc> will start OR. You're sitting at the train driver's chair in a diesel locomotive. Since the engine has already been started, you can hear it now.
  • Press key <2> to see the locomotive from the outside. Press key <h> twice for switching the headlights on and keys <Shift + h> for switching them off again. See, you are really in command of this locomotive. Arrow keys can be used to move around in this view. As is the mouse in combination with it's right key. And also try the mouse wheel. Press key <1> to get you back in the drivers seat.
  • Press key <F4> to have the "Track Monitor" window popup. Drag it to a convenient place. OR will remember the place for all such windows between startups. This window gives track info, such as signals, speed limit, etc.
  • Press key <F5> to have the "Train Driving Info" popup. Engine is running. Important lines for now are Speed, Direction and Train Brake.
  • It is possible to handle the throttle and the brake by clicking the mouse pointer on the appropriate handle on the screen. But it's difficult to do this properly. So recommended is using the keys on the keyboard.
  • Let's switch the direction to forward by pressing the key <w>. See the direction change in the Train Driving Window. Press key <s> for Neutral and Reverse.
  • Press and hold down key <;> until the Train brake is released. All those numbers in bar and psi in the "Train Driving Info" popup are not that important now. Only when BC (Brake Cylinder) has reached 0 bar the brakes are released and you are ready to go.
  • Notice the green signal outside, in the UK on the left side of the track. Press key <d> twice to have the throttle moved to 25%. It's rolling... Speed limit is 20 mph. When the actual speed reaches this limit the tracks in the Track Monitor window color orange as a warning. When seriously over speeding they color red. Key <a> is for throttling down.
  • Press key <4> to act as a trainspotter who very fast hops from place to place. Key <3> gives you a view of the end of the train.
  • It's a bit boring to run all the way to Glasgow. Therefore suggested is to press the keys <Alt + F4> to stop OR at some time.
  • You'll get back into the menu. Go to the drop down list Activiy and choose "0930 Edingburgh-Glasgow Queen Street". And press the button "Start".
  • Next to <F4> and <F5> press key <F10>. You will also be asked to do so by an "Activity Events" window. You will now see the "Next Station" with the train schedule you have to meet as a driver. It's not that easy. When you want to see how this is done switch to autopilot mode by pressing key <Alt + a>. It's a toggle switch, pressing again switches autopilot mode off. Autopilot mode is acting a bit nervous, but anyway it does it's job.

You have now seen how to run Open Rails. There is plenty more to explore: electric locomotives, steam locomotives, turn around's, demanding schedules, shunting wagons...

Have fun!
